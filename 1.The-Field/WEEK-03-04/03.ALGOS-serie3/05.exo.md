# Exercise.

---

Create a function that converts Celsius to Fahrenheit and vice versa.

#### Notes

- Round to the nearest integer.
- If the input is incorrect, return "Error".
- Look on Google how to convert F to C and vice-versa

#### Expected results

```
convert("35°C") ➞ "95°F"

convert("19°F") ➞ "-7°C"

convert("33") ➞ "Error"
```
